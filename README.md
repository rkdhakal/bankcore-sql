# BankCore SQL Project

A comprehensive SQL project built on a banking domain to practice and master intermediate to advanced SQL concepts.

## Concepts Covered
- Complex JOINs & Subqueries
- Aggregations, GROUP BY, HAVING
- Common Table Expressions (CTEs)
- Window Functions (ROW_NUMBER, RANK, LAG, LEAD, running totals)
- Stored Procedures & Triggers
- Views & Query Optimization
- Transactions & ACID principles

## Project Structure

| Folder | Purpose |
|--------|---------|
| `schema/` | Table creation scripts and database design |
| `data/` | Sample data insert scripts |
| `queries/` | SQL practice queries organized by phase |
| `procedures/` | Stored procedures, triggers, and views |

## Database: MySQL

## Phases
1. Schema Design
2. JOINs + Subqueries
3. Aggregations + CTEs
4. Window Functions
5. Stored Procedures + Triggers
6. Views + Query Optimization
7. Capstone — Real analyst-style business questions
